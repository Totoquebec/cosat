Compteur de visites

Il s'agit d'un compteur de visites du site à placer en page d'accueil réalisé en php.

Ce compteur ne s'incrémente qu'au premier chargement de la page d'accueil.
Nécessite la librarie GD.Voir si celle ci est bien présente avec le script gd.php
Vous pouvez changer ou renommr les images (0.png à 9.png, im0.gif ou im0.jpg)
livré avec le script, l'essentiel est qu'il y ait les 10 images et que leur nom se terminent
par 0 à 9 suivi de l'entension png, jpg, ou gif.
exemple: mon_image_0.jpg, mon_image_1.jpg ,..., mon_image_9.jpg
exemple: image0.gif, image1.gif ,..., image9.gif
exemple: 0.png, 1.png ,..., 9.png

Le script gd.php précise aussi si les types png, jpg et png sont supportés.

Le script compteur.php est appelé pour la réalisation du compteur.Il contient 2 paramètres:

  • le chemin et le nom du fichier texte contenant le nombre de visites par défaut compteur.txt
    (exemple: ./compteur.txt)
  • le chemin et le nom d'une image représentant un des 10 chiffres
    (exemples: ./png/0.png ou ./image0.jpg ou ./gif/img0.gif )

Placer une image dans votre page d'accueil et lui indiquer comme source le script compteur.php
ou coller directement dans le code source de la page d'accueil ceci: <img src="compteur.php">
